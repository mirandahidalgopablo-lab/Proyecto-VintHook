Two Engines, One a Spare

Buscar en Vinted no es una petición: son decenas, contra varios países y varias páginas de resultados a la vez. La búsqueda turbo detecta en arranque si aiohttp está instalado. Si lo está, lanza todas las peticiones en un único bucle de eventos. Si no, cae a un pool de hilos y nadie se entera.

02 / EstadisticaStatistics

El Precio Justo

What It's Actually Worth

Para saber si un anuncio es una oportunidad hay que saber cuánto vale de verdad. La media miente: basta con un artículo puesto a 900 euros por error para arruinarla. Se limpian los extremos por rango intercuartílico y se trabaja con la mediana.

To know whether a listing is a deal you first need to know what the thing is really worth. The mean lies: one item mispriced at 900 euros is enough to wreck it. Outliers get cut with the interquartile range and the median does the work.

app.pyclean_outliers_iqr
def clean_outliers_iqr(values):
    """Remove outliers using Interquartile Range method."""
    if len(values) < 4:
        return values
    s = sorted(values)
    n = len(s)
    q1 = s[n // 4]
    q3 = s[(3 * n) // 4]
    iqr = q3 - q1
    lower = q1 - 3.0 * iqr
    upper = q3 + 3.0 * iqr
    cleaned = [v for v in values if lower <= v <= upper]
    return cleaned if cleaned else values

The multiplier is 3.0, not the textbook 1.5. Second-hand clothing genuinely spreads that wide, and an aggressive cut would throw out legitimate listings. Only the absurd gets dropped.

03 / AutoPilot

El Hilo Que Nunca Duerme

The Thread That Never Sleeps

Un solo hilo daemon atiende a todos los usuarios con AutoPilot activo. Despierta cada diez segundos, mira a quién le toca ciclo según su intervalo, lo ejecuta y vuelve a dormirse. Un fallo en el ciclo de un usuario se registra y se sigue: nunca tumba el worker.

One daemon thread serves every user with AutoPilot on. It wakes every ten seconds, checks whose cycle is due based on their interval, runs it, and goes back to sleep. A failure in one user's cycle is logged and skipped: it never takes the worker down.

Tic de diez segundosTen-second tick
El bucle no gira en vacío quemando CPU: duerme diez segundos entre pasadas y reparte los ciclos según el intervalo que cada usuario haya configurado.
The loop doesn't spin hot burning CPU: it sleeps ten seconds between passes and schedules cycles by whatever interval each user configured.
Aislamiento por usuarioPer-user isolation
Cada ciclo va envuelto en su propio try. Si la sesión de alguien caduca o la API responde raro, ese usuario ve el error en su log y los demás siguen operando.
Every cycle is wrapped in its own try. If someone's session expires or the API answers oddly, that user sees the error in their log and everyone else keeps running.
Techo de accionesAction ceiling
Hay un límite duro de acciones por ciclo, configurable por variable de entorno. Si la lista de candidatos lo supera, se corta y se avisa. Pasarse de ritmo es la forma más rápida de que te bloqueen.
There's a hard cap on actions per cycle, set by an environment variable. If the candidate list exceeds it, it's truncated and logged. Going too fast is the quickest way to get blocked.
Estado en memoriaIn-memory state
Última ejecución, próxima ejecución y estado viven en un mapa en memoria que el panel consulta por API. No se toca la base de datos para pintar un contador.
Last run, next run and status live in an in-memory map the dashboard polls over the API. The database isn't touched just to paint a counter.

Escrituras Que Se Pueden Repetir

Writes You Can Repeat

Un worker que reintenta va a escribir lo mismo dos veces tarde o temprano. Todas las escrituras son idempotentes por diseño: ON CONFLICT DO UPDATE cuando el dato nuevo manda, ON CONFLICT DO NOTHING cuando el primero que llegó tenía razón.

A worker that retries will write the same thing twice sooner or later. Every write is idempotent by design: ON CONFLICT DO UPDATE when the new value wins, ON CONFLICT DO NOTHING when whoever got there first was right.

09 / SeguridadSecurity

Lo Aburrido, Bien Hecho

The Boring Parts, Done Right

Nada de lo que viene es original, y ese es exactamente el punto. La seguridad de una aplicación que mueve dinero y sesiones ajenas no se gana con una idea brillante: se gana no cometiendo los diez errores de siempre. Aquí están, uno por uno, y cómo están cerrados.

None of what follows is original, and that is exactly the point. Security in an app that handles money and other people's sessions isn't won with a clever idea: it's won by not making the same ten mistakes as everyone else. Here they are, one by one, and how each is closed.

Inyeccion SQLSQL injection
Hay 92 consultas a la base de datos en el proyecto. Cero usan f-strings o concatenación: todas pasan los valores como parámetros y es el driver quien los escapa. No hay una sola grieta por donde colar una comilla.
92 database queries across the project. Zero use f-strings or concatenation: every value is passed as a parameter and the driver does the escaping. There isn't a single crack to slip a quote through.
ContrasenasPasswords
Nunca se guarda una contraseña. Se guarda su hash, generado con Werkzeug, con sal propia por usuario. Ni siquiera desde dentro de la base de datos se pueden leer. Mínimo ocho caracteres al registrarse y al resetear.
A password is never stored. Its hash is, generated with Werkzeug, individually salted per user. Not even from inside the database can they be read back. Eight characters minimum on signup and on reset.
Secuestro de sesionSession hijacking
La cookie de sesión va firmada criptográficamente, es inaccesible desde JavaScript (HttpOnly), no viaja nunca por HTTP plano (Secure) y no se envía en peticiones cruzadas (SameSite=Lax). Un XSS no te la roba y un CSRF no la usa.
The session cookie is cryptographically signed, unreachable from JavaScript (HttpOnly), never travels over plain HTTP (Secure), and isn't sent on cross-site requests (SameSite=Lax). An XSS can't steal it and a CSRF can't use it.
XSS
Las plantillas escapan por defecto todo lo que se inyecta en ellas. Un título de Vinted que traiga una etiqueta <script> se pinta como texto, no se ejecuta. Y el contenido viene de terceros, así que esto no es teórico.
Templates escape everything injected into them by default. A Vinted title carrying a <script> tag renders as text; it doesn't run. And that content comes from third parties, so this isn't hypothetical.

Fuerza brutaBrute force
Diez intentos por IP, seis por cuenta
Ten attempts per IP, six per account
Un atacante puede probar diez contraseñas cada cinco minutos desde una IP. Pero si reparte el ataque entre mil IPs para esquivar ese límite, se topa con el segundo: seis intentos por hora contra la misma cuenta, venga de donde venga. Diluir el origen ya no sirve de nada. Y la ventana es deslizante, no un contador que se pone a cero a la hora en punto, así que tampoco vale esperar al cambio de hora para disparar en ráfaga.
An attacker can try ten passwords every five minutes from one IP. But spread the attack across a thousand IPs to dodge that limit and they hit the second one: six attempts per hour against the same account, wherever they come from. Diluting the source buys nothing. And the window slides, it isn't a counter that zeroes on the hour, so waiting for the rollover to fire a burst doesn't work either.

Every AI call costs real money

Twenty image analyses and thirty listing generations per user per hour is plenty to work with, and far too little to ruin anyone.
La IP realThe real IP
Detras del proxy, todos parecen la misma persona
Behind the proxy, everyone looks like one person
Este es el detalle que convierte un limitador en una caída de servicio si se pasa por alto: detrás de un proxy inverso, todas las peticiones llegan desde 127.0.0.1. Un límite por IP se volvería un límite global sin avisar, y el primer atacante dejaría fuera a todos los usuarios legítimos. Por eso la IP se resuelve leyendo la cabecera que escribe el proxy, confiando en un único salto para que nadie pueda falsificarla desde fuera.
This is the detail that turns a rate limiter into an outage if you miss it: behind a reverse proxy, every request arrives from 127.0.0.1. A per-IP limit would quietly become a global limit, and the first attacker would lock out every legitimate user. So the IP is resolved from the header the proxy writes, trusting exactly one hop so nobody can forge it from outside.
CabecerasHeaders

Endpoints aren't decorated one by one: a single after_request stamps them all. nosniff stops the browser guessing content types, SAMEORIGIN blocks iframe clickjacking, Referrer-Policy keeps internal paths from leaking off-domain, Permissions-Policy switches off camera, microphone and geolocation because the app doesn't use them, and HSTS with preload forces HTTPS before the very first request.
